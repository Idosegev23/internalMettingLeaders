# מסמך התנעה - לידרס 2025

טופס פנימי למילוי בפגישת בריף פנימית לאחר קבלת הבריף המלא מהלקוח.

## תכונות

- **חיפוש חכם לאנשי צוות** - חיפוש מתוך קובץ CSV עם autocomplete
- **בחירת מספר משתתפים** - אפשרות לבחור מספר אנשים לשדות רלוונטיים
- **שליחה אוטומטית** - כל הנתונים כולל שמות באנגלית ומיילים נשלחים לוובהוק
- **responsive מלא** - מותאם למובייל ולדסקטופ

## התקנה

```bash
npm install
```

## הרצה במצב פיתוח

```bash
npm run dev
```

האתר יפתח בכתובת: http://localhost:3001

## קובץ אנשי הצוות

הקובץ `Contacts - שמות מלא.csv` מכיל את רשימת כל אנשי הצוות עם:
- שם באנגלית
- שם בעברית
- כתובת מייל

הטופס מחפש בקובץ זה ומאפשר לבחור אנשים בצורה נוחה.
